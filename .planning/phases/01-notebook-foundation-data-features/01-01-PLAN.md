<task type="auto">
  <name>Task 2: Create data loading and preprocessing module</name>
  <files>src/data.py</files>
  <action>
1. Create `src/data.py` with the following exports:

   **`RVL_CDIP_LABELS`**: A list of the 16 RVL-CDIP class names in order:
   ["letter", "form", "email", "handwritten", "advertisement", "scientific_report", "scientific_publication", "specification", "file_folder", "news_article", "budget", "invoice", "presentation", "questionnaire", "resume", "memo"]

   **`get_transform()`**: Returns a `transforms.Compose` pipeline:
   - `transforms.Resize((224, 224))`
   - `transforms.Grayscale(num_output_channels=3)` -- replicates grayscale to 3 channels for ResNet-50
   - `transforms.ToTensor()`
   - `transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])` -- ImageNet stats

   **`load_rvl_cdip(config: Config)`**: Loads the dataset based on config.mode:
   - If `config.mode == "sample"`:
     - Use `load_dataset(config.dataset_name, streaming=True)`
     - For each split (train, validation, test), use `.take(config.sample_size)` and convert to list
     - Return a dict: `{"train": list, "validation": list, "test": list}`
     - Each item in the list is a dict with keys "image" (PIL Image) and "label" (int)
   - If `config.mode == "full"`:
     - Use `load_dataset(config.dataset_name, cache_dir=str(config.hf_cache_dir))`
     - Return the dataset dict directly (HuggingFace DatasetDict)

   Important implementation details:
   - Do NOT use `trust_remote_code=True` unless explicitly needed (RVL-CDIP does not need it)
   - Streaming returns IterableDataset -- `.take()` returns IterableDataset, so convert to list for sample mode
   - Full mode returns DatasetDict which supports indexing natively
   - Print split sizes after loading: `f"Loaded {split}: {len(data)} samples"`
   - For streaming mode, print after converting to list since IterableDataset has no len()

2. Add a `display_samples(data, n=10)` function:
   - Takes a list of sample dicts (from load_rvl_cdip output)
   - Uses matplotlib to display n images in a 2-row grid
   - Shows class label text (from RVL_CDIP_LABELS) as title on each subplot
   - Uses `cmap='gray'` since images are grayscale
   - Returns the matplotlib figure (for notebook display)

3. Add a `apply_transform(samples, transform)` function:
   - Takes a list of sample dicts and a torchvision transform
   - Returns list of dicts with "image" replaced by transformed tensor [3, 224, 224] and "label" kept as int
   - This is for batch preprocessing before feature extraction
  </action>
